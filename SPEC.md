# ワードスクランブルゲーム【無料】無料ゲーム - 技術仕様書

## 概要

**サービス名**: Word Scramble Game
**バージョン**: 1.1.0
**更新日**: 2026-06-03
**URL**: https://appadaycreator.com/word-scramble-game/

バラバラになった文字を並べ替えて単語を作るゲーム。英語・日本語対応。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: 以下の情報を自動保存
  - 各カテゴリの最高スコア
  - 前回選択していたカテゴリ
  - 総プレイ数
  - プレイ統計情報（平均スコアなど）
- **クリア**: ブラウザのサイトデータ削除で初期化
- **ストレージキー**: `wordScrambleGameData`

## 機能一覧（v1.1.0）

### ゲーム機能
- **マルチカテゴリ対応**: 英語（基本・動物・食べ物）/ 日本語（カタカナ）の4種類
- **10問1セット**: スクランブルされた文字を並べ替えて正解を目指す
- **ヒント表示**: 各問題にカテゴリ別のヒント付き
- **スコア管理**: リアルタイムスコア表示、最高スコア自動更新

### ユーザー体験
- **自動復元**: 前回選択していたカテゴリを自動記憶
- **統計情報表示**: 総プレイ数・最高スコア・平均スコアをリアルタイム表示
- **フィードバック**: 正解/不正解を色分け表示、自己ベスト更新時に通知

### キーボード操作（v1.1.0 追加）
- **数字キー（1-9）**: 文字タイル選択
- **Enter**: 答え合わせ実行
- **Backspace**: 最後の文字を戻す
- **Space**: 回答欄クリア
- **Escape**: スキップ

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）
- ローカルストレージ（localStorage）による永続化

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: ワードスクランブルゲームは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
